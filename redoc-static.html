Enterprise Authentication Service API (1.0.0)

Download OpenAPI specification:

This service handles user authentication and authorization for the enterprise application.

Authenticate a user

Authenticates a user with an email and password. A successful login returns an access token and sets a secure HttpOnly refresh token cookie.

header Parameters
X-Device-Id
required
string
Example: abcdef1234567890

Unique identifier for the user's device.

Request Body schema: application/json
required
email
required
string

The user's email address.

password
required
string

The user's password.

Responses

Local development server

http://localhost:3000/api/auth/login

Request samples

  • Payload
Content type
application/json
{
  • "email": "victoroluborode@example.com",
  • "password": "securepassword123"
}

Response samples

  • 200
  • 401
Content type
application/json
{
  • "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VySWQiOiJ2aWN0b3JvbHVib3JvZGUiLCJpYXQiOjE2ODcyMDA2NDgsImV4cCI6MTY4NzIwNDI0OH0.SFLy7gU5T1hS4fR5jK8y8f8JvL2fQv4oD3C8gL8q",
  • "message": "Login successful",
  • "user": {
    • "id": "65e23c7c2518e3a2410a0e70",

    • "fullname": "User Name"
    }
}
